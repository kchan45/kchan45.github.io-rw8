Building a Split Mechanical Keyboard

This keyboard was built using the kit made by Brian Low. The switches and keycaps were obtained from MKUltra Corporation. The keyboard case was 3-D printed using Hatchbox wood filament on a Creality Ender 3 v2.

The files for printing this case are here, though I did not use the gasket mount technique, just opting for rubber feet to lift the keyboard to fit the case.
