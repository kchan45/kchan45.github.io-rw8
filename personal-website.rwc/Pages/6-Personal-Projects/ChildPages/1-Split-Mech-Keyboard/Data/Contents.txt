# Building a Split Mechanical Keyboard

This keyboard was built using the kit made by [Brian Low](https://choc.brianlow.com). The switches and keycaps were obtained from [MKUltra Corporation](https://mkultra.click). The keyboard case was 3-D printed using Hatchbox wood filament on a Creality Ender 3 v2.
